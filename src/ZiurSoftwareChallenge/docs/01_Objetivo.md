# 01 - Objetivo

Documento que explica el objetivo del proyecto actual.

Objetivo general

- Construir una aplicación Blazor Web App (.NET 9) que consuma una API REST y muestre los datos en una grilla.
- La aplicación actual consume la API real y muestra el estado de la conexión en la UI. Está preparada para usar `HttpClient` y configuración segura de tokens.

Alcance de esta documentación

- Documentar únicamente el estado y los artefactos presentes en este repositorio en su forma actual (no describe proyectos anteriores ni versiones previas removidas).
